# システム構成・概要

## システム構成

以下の通信経路で GitLab へのアクセスを可能にする。

```text
Public Gateway Host
  ↓ WireGuard
Private GitLab Host
  ↓
GitLab API
```

## Public Gateway Host

- ホスト: インターネット上に公開している Linux サーバー
- WireGuard: 10.20.30.1

## Private GitLab Host

- ホスト: ローカル環境で GitLab を運用している Linux サーバー
- WireGuard: 10.20.30.2

## 構成要素

- nginx
- Public Gateway
- local-gitlab-gateway-private
- GitLab
- WireGuard
