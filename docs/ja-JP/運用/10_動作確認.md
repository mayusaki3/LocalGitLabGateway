# 動作確認

## Public Gateway

```bash
curl -i http://127.0.0.1:8080/health
```

## Private Bridge

```bash
curl -i http://127.0.0.1:8081/internal/health
```

## WireGuard 経由

```bash
curl -i http://10.20.30.2:8081/internal/health
```

## nginx 経由

```bash
curl -i http://127.0.0.1/api/health
```

## GitLab Login

```bash
curl -i http://127.0.0.1/gitlab/users/sign_in
```
