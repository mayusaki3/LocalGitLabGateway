# トラブルシュート

## nginx

```bash
sudo nginx -t
sudo nginx -T
systemctl status nginx --no-pager
```

## WireGuard

```bash
systemctl status wg-quick@wg0 --no-pager
sudo wg show
```

## Public Gateway

```bash
systemctl status local-gitlab-gateway-public --no-pager
journalctl -u local-gitlab-gateway-public -n 100 --no-pager
```

## Private Bridge

```bash
systemctl status local-gitlab-gateway-private --no-pager
journalctl -u local-gitlab-gateway-private -n 100 --no-pager
```

## GitLab Reverse Proxy

```bash
curl -k -i https://10.20.30.2/gitlab/users/sign_in
```

## login 422

確認項目:

- X-Forwarded header
- proxy_pass path
- Host header
- Cookie
- CSRF
